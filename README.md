# SemEval 2026 Task 12

This repository contains the code, data, and results for the SemEval 2026 Task 12 submission.

## Repository Structure

```
SemEval2026-task12/
├── data/
│   ├── raw/          # Raw, unprocessed datasets
│   └── processed/    # Preprocessed/cleaned data
├── src/              # Source code and scripts
├── notebooks/        # Jupyter notebooks for analysis and experiments
├── results/          # Model outputs, evaluation results
└── README.md
```

## Getting Started

### Uploading Your Local Files

To upload your local project files (e.g., from `C:\Users\Lenovo\OneDrive - Newcastle University\SemevalTask12`) to this repository, follow these steps:

#### Option 1: Using Git (Recommended)

1. **Clone this repository** to your local machine:
   ```bash
   git clone https://github.com/MingkaiW/SemEval2026-task12.git
   ```

2. **Copy your files** from your local folder into the cloned repository folder. Place files in the appropriate subdirectory:
   - Source code → `src/`
   - Raw datasets → `data/raw/`
   - Processed data → `data/processed/`
   - Notebooks → `notebooks/`
   - Results/outputs → `results/`

3. **Stage, commit, and push** your changes:
   ```bash
   cd SemEval2026-task12
   git add .
   git commit -m "Add project files from local SemevalTask12 folder"
   git push origin main
   ```

#### Option 2: Using GitHub Web Interface

1. Navigate to this repository on GitHub.
2. Click **"Add file"** → **"Upload files"**.
3. Drag and drop your files or click to browse and select them.
4. Add a commit message and click **"Commit changes"**.

#### Option 3: Using GitHub Desktop

1. Download and install [GitHub Desktop](https://desktop.github.com/).
2. Clone this repository via GitHub Desktop.
3. Copy your project files into the local repository folder.
4. Commit and push the changes through the GitHub Desktop interface.
